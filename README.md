Completion Trie
===============

Completion Trie implementation based on Tahseen Ur Rehman and Javid Jamae's work
available [here].

This implementation relies on B.-J. Hsu and G. Ottaviano's paper *Space Efficient Data
Structures from Top-k Completion*.
